An Open Achievement Standard for On-Chain Games

2026年4月28日 · 約5分
EffectStream

Steam has achievements. Xbox has Gamerscore. PlayStation has Trophies. But on-chain games have no standard way to track player accomplishments across games. We built one — an open standard that any EffectStream game can implement, plus a portal where players can see all their achievements in one place.
